<ContentText>Hacktoberfest adalah acara tahunan di seluruh dunia yang diadakan oleh Digital Ocean selama bulan Oktober. Acara ini mendorong open source developer untuk berkontribusi pada repositori melalui permintaan pull request (PR)</ContentText>
<ContentText>Dulu pertama kali saya tahu informasi mengenai hacktoberfest adalah ketika saya masih semester awal masa kuliah kira-kira tahun 2019 dari teman komunitas. Disitu sebenarnya sudah banyak mereka yang mulai ikut kontribusi di event ini. Dan bahkan komunitas yang ada di surabaya dan malang mengadakan agenda opening hacktoberfest ini untuk memberi wawasan dulu ke para developer pemula untuk ikut berpartisipasi. Tetapi di situ saya masih belum percaya diri untuk bisa kontribusi ke open source, mengingat saya masih belum mempunyai skill yang mumpuni untuk memberikan kontribusi maupun membantu solve problem yang ada lewat code.</ContentText>
<ContentText>Dan di tahun depannya 2020, saya memutuskan untuk memfokuskan diri ke bidang frontend web development karena saya rasa sejak sama SMK sudah mempunyai basic javascript, jadi seandainya fokus ke frontend ini sangat mudah untuk dikembangkan ilmu nya, dan tentunya bisa lebih cepat terjun ke open source.</ContentText>
<ContentText>Di tahun terkhir saya kuliah 2022, saya sudah ada mulai ada bekal untuk kontribusi, saya memutuskan untuk membuka website hacktoberfest, dan saya mencoba untuk mendaftarkan diri, membaca rules contribution. Dan setelah itu saya mencoba mencari repositori di github yang berlabel hacktoberfest. Dan saya mulai kontribusi awal saya di repository learn programming komunitas PasuruanDev, yaitu repository mengenai materi pembelajaran tentang berbagai bahasa pemrograman seperti javascript, golang, typescript, python, c, lua yang dibuat oleh komunitas developer di Pasuruan, Jawa Timur. Saya kontribusi di bahasa JavaScript dan TypeScript, mengisi materi tentang Loop, Function, Enum, Interface, dll.</ContentText>
<LargeImages classnames="h-[250px]" description="Kontribusi materi javascript menggunakan markdown" image="add-material.png" />
<ContentText>Setelah membuat pull request ke maintaner repository nya, ternyata pull request accept / diterima, dan maintaner memberikan label hacktoberfest agar pihak hacktoberfest bisa tahu dan mengecek pull request tersebut. Sehingga langkah berikut nya tinggal menunggu keputusan dari pihak hacktoberfest, diseleksi pull request yang sudah saya berikan ke komunitas tersebut sudah memenuhi persyaratan atau belum.</ContentText>
<LargeImages classnames="h-[250px]" description="Top Kontributor di Repository PasuruanDev" image="stats.png" />
<ContentText>Durasi waktu tunggu proses seleksi tersebut kurang lebih 5 harian. Jadi sembari nunggu di acc atau di tolak, kita bisa mencoba kontribusi lagi di repositori yang lain, jadi disitu saya coba search lagi repository yang dia berlabel hacktoberfest. Dan begitu seterusnya.</ContentText>
<ContentText>Hacktoberfest sendiri memberikan apresiasi tiap individu sesuai jumlah kontribusi yang telah diberikan, contoh nya kalau kita berhasil kontribusi 4 (maksimal) open source kita akan mendapatkan SWAG spesial dari hacktoberfest yang isi nya kaos limited edition hacktoberfest 2022, stiker dari berbagai sponsor seperti docker, appwrite, devtron, github, dll, dan beberapa credit untuk bisa diclaim.</ContentText>
<ContentText>Dan bagusnya dalam satu bulan itu saya sudah mendapatkan 7 pull request yaitu ke repository (Eropa, Asia, India dan Indonesia) dari 4 maksimum contribution, dan tiap pull request juga sudah di accept 5, dan 2 nya gagal karena yang 2 itu repositori nya sudah tidak diakui / bisa disebut deprecated sudah lama dan tidak lolos seleksi hacktoberfest. Tapi dari 5 itu sudah sangat cukup karena sudah melewati batas maksimal 4 kontribusi tadi.</ContentText>
<ContentText>Dan bulan depan nya saya mendapatkan email bahwa saya mendapatkan badge limited edition dari hacktobefest dan form untuk memilih SWAG warna kaos dll.</ContentText>
<ContentText>Setelah mengisi form, kira kira menunggu shipping ( pengiriman dari frankfurt germany ), perlu waktu agak lama yaitu 2-4 bulan mulai dikirim november 2022, tergantung lokasi negara nya. Dan paket SWAG sampai di saya Kediri, Indonesia di awal Februari 2023.</ContentText>
